AWS OpenTelemetry ServiceEvents Instrumentation
================================================

Installation
------------

::

    pip install aws-opentelemetry-serviceevents


ServiceEvents instrumentation for the AWS Distro for OpenTelemetry Python. Provides deep
observability capabilities including function-level invocation metrics, HTTP endpoint
performance tracking, and automated error investigation on failures. Registers as an
OpenTelemetry instrumentor and is bundled with Application Signals.
