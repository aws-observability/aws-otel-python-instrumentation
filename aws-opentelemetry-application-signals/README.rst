AWS OpenTelemetry Application Signals
======================================

Installation
------------

::

    pip install aws-opentelemetry-application-signals


Application Signals components for the AWS Distro for OpenTelemetry Python. Provides the
span processors, sampler, and metric-attribute generation used to derive
service/dependency metrics and propagate metric-correlation attributes. Consumed by
``aws-opentelemetry-distro`` and ``aws-opentelemetry-serviceevents``.
